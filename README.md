**安装步骤**:
1.下载并安装 Love2d Snippets 插件
2.将 json 文件复制到 C:\Users\UserName\.vscode\extensions\pixelwar.love2dsnippets-2.0.0\snippets 文件夹下。不同用户的路径会有些许差异，可以使用 Everything 工具快速定位
3.重启 VSCode
