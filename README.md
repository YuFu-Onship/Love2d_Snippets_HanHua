1.下载并安装Love2d Snippets插件
2.将json文件复制到C:\Users\UserName\.vscode\extensions\pixelwar.love2dsnippets-2.0.0\snippets文件夹下即可,不同用户的路径会有些许差异,可以使用everything工具快速定位到目标路径
3.重启vscode
